## Additional Notes

### Key Technologies (for reference)
- **Backend**: Flask, Python, Gunicorn
- **Database**: PostgreSQL, SQLAlchemy, Flask-Migrate
- **Frontend**: HTML5, CSS3, Bootstrap 5, Jinja2
- **APIs**: Azure Computer Vision, Spoonacular API
- **DevOps**: Docker, Docker Compose, Multi-container architecture
- **Security**: Flask-Login, Bcrypt, OAuth (Google), CSRF Protection
- **Architecture**: Load-balanced (3 instances), Network segregation

### Project Highlights
- AI-powered ingredient recognition from photos
- Smart recipe search based on available ingredients
- User authentication with Google OAuth and traditional email/password
- Secure password reset with email verification
- Cookie consent management (GDPR compliant)
- Google Analytics integration
- Responsive design for all devices
- Load-balanced architecture with fault tolerance
- Docker network segregation for security
